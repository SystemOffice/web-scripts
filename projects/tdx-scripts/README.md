# tdx-scripts-wip

Adds customizations to the TDX Client Portal. The .js files can be embedded into a TDX Client Portal using the Client Footer in the portal Settings. Using the jsdeliver CDN adds a layer of reliability and you can automatically minimize the code, i.e. 

<script src="https://cdn.jsdelivr.net/gh/SystemOffice/web-scripts@main/projects/tdx-scripts/tdx.min.js" defer>
</script>
